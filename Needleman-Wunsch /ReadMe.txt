needleman.pl:
_ A script that simulates the Needleman-Wunsch global alignment algorithm without gaps using dynamic programming relations. 

needleman_gaps.pl:
_ This is the version of the Needleman-Wunsch alignment algorithm incoporates gaps handling.

extend_needleman.pl:
_ This version used the Needleman-Wunsch algorithm to compute the longest common subsequence and the average LCS length.
